Projects

Bridge Break Detection System

This project was inspired by the tragic Mahad bridge collapse incident where many lives were lost due to sudden structural failure. To address this problem, we designed an early warning safety system capable of detecting bridge collapse and immediately alerting nearby vehicles and authorities. When a bridge failure is detected, the system activates a red signal and siren near the bridge to warn approaching vehicles while simultaneously sending notifications to responsible authorities through a dedicated mobile application. The application was developed using MIT App Inventor to deliver instant alerts to the concerned departments and organizations. This project focuses on improving public safety infrastructure by providing real-time alerts that can help prevent accidents and reduce potential loss of life. The project received recognition and secured 3rd rank in the college project competition.
